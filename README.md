# Archetypal AI Homebrew Tap

Official Homebrew tap for [Archetypal AI](https://github.com/archetypal-ai/archetypal-ai).

## Install

```bash
brew tap archetypal-ai/tap
brew install archetypal
```

## Update

```bash
brew upgrade archetypal
```

## Uninstall

```bash
brew uninstall archetypal
brew untap archetypal-ai/tap
```
